01 Jan 0001

First, I installed a few Visual Studio extensions that that leverage Roslyn and acts as code analyzers (in addition to the extensions I normally use)

  • AsyncFixer
  • C# Essentials
  • Exception Analyzer
  • Refactoring Essentials

And MVVM-Tools to replace the Navigate between XAML View and ViewModel Resharper plugin.

Then I did some keyboard shortcuts remapping to keep the Resharper shortcuts I am used to and make them invoke native Visual Studio functions

  • Ctrl+, -> Ctrl+T (navigate to any)
  • Ctrl+. -> Alt+Enter (show quick fix)
  • Go to implementation -> Ctrl+F12
  • MVVM Tools -> Ctrl+Alt+F7

I did not find a good replacement for the Resharper unit test runner functionality, but for Windows Phone projects it is broken most of the the time (yes, I reported it to JetBrains a few times already).
